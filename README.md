# SignNet
Jupyter-based project, developed in python and responsible for scanning &amp; detecting Germany's warning signs (GTS-rb) fetched from a sample Kaggle database. Using CNN, the developed program, reliably detects the shape and meaning of the signs with accuracy of 95%+.
Download the database from: https://www.kaggle.com/datasets/meowmeowmeowmeowmeow/gtsrb-german-traffic-sign
